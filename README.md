# 百鬼夜行 HYAKKI YAGYO

陰陽師となり、百鬼夜行の夜を夜明け(30分)まで生き延びるサバイバーアクション。
**Vampire Survivors の隠れ仕様を完全リバースエンジニアリングした機能クローン**
(テーマ・素材・名称・音声は100%オリジナル、素材ファイル0個=全て手続き生成)。

- 🎮 遊ぶ: `npm run dev` → http://localhost:5199/play/ (ランディングは `/`)
- 📜 RE文書: [MECHANICS.md](MECHANICS.md) — 原作の仕組み→本実装の対応表(式・テーブル・アルゴリズム、全てテストが実行)

## 開発

```bash
npm install
npm run dev        # 開発サーバ (5199)
npm test           # 58 tests
npm run typecheck  # TS strict
npm run build      # 型検査+本番ビルド (dist/)
npm run bot        # バランス掃引(キャラ×シード×投資、ゲート付き)
```

デバッグURL: `/play/#debug`(perf HUD)・`/play/#perf`(負荷ゲート)・`/play/#sprites`(アトラス一覧)。
ゲーム内チート: `window.__hyakki.setTimeScale(8)`。

## アーキテクチャ(要点)

- **sim/presentation分離**: `src/game/sim` はDOM/描画/時計/Math.random禁止(テストが強制)。
  ボット・CI・リプレイは全部この境界の恩恵
- **固定60Hzタイムステップ**+補間描画。ドラフト中はsim自体が凍結(原作準拠)
- **アーキタイプ別プール**+空間ハッシュ+単一アトラスblit — GCゼロ・500体でも余裕
- 敵ごとの`hitUntil[weaponSlot]`ゲートで持続武器の再ヒット間隔を再現(原作のhitbox delay)

## バランス検証(ヘッドレスボット)

`npm run bot` が {キャラ×シード×護符投資} を全ラン実行し、実行可能ゲートで判定:

- 初期セーブ: 中央値 **15.3分**で死亡(6〜17分帯域)
- 50%投資: 中央値 **20.2分**(+4.9分 — メタ進行の効果を定量証明)
- 夜明け到達はボットでは0%(単純キーターのため)。人間の腕と進化ビルドで到達する設計
